Flexbox

Um pequeno resumo sobre Flexible Box Layout module. Para fixação de conteúdo aprendido durante a realização do curso de HTML5 e CSS3 do Curso em Vídeo

O que são Flexbox

Flexible Box Layout module ou resumindo Flexbox, foram projetadas como um modelo de layout unidimensional. Isso quer dizer que os elementos são configurados em uma dimensão por vez, sendo em linhas ou em colunas. Foram criadas para para organizar os elementos dentro de uma interface.

Abaixo veremos os princípios básicos e exemplos práticos, mas antes é preciso entender os eixos no flexbox.

Definindo um item com flexbox

Antes de tudo é preciso entender como "ativar" o flexbox e qual vai ser o comportamento dos elementos. Os itens que serão dinamicamente configurados precisam de um pai. Mas quem é esse pai? A resposta é: depende!

Para simplificar as coisas vamos imaginar que temos uma div e dentro dessa div temos elementos p. O pai é a div e os elementos p são seus filhos. Dito isso agora podemos entender o começo da configuração com flexbox

Para começar é necessário atribuir um display flex ao pai:


            <style>
                div#container {
                    display: flex;
                }
            </style>

Entendido isso podemos começar a entender os eixos no flexbox.

Os eixos no flexbox

O flexbox possui dois eixos em que podemos configurar os elementos: eixo principal (main axis) e eixo transversal (cross axis).

O main axis é quem dita como as configurações vão ser dispostas, por isso não se deve entender o flexbox com comandos de direção padrão como: cima, baixo, direita e esquerda.

Além dessas duas configurações, ainda temos que ter em mente que cada eixo possui um início e um fim. Definidos por main-start e main-end, ou então por cross-start e cross-end.

Empacotamento e fluxo no Flexbox

São usadas duas definições para alterar o comportamento do pai. O flex-wrap e o flex-direction. O flex-wrap, como o próprio nome sugere wrap no inglês significa "embrulhar" e direction significa "direção".

No Flexbox o padrão de empacotamento é o nowrap. Isso significa que o conteúdo presente no contâiner pai não será deslocado, ou quebrado. Porém é possível alterar essa configuração para wrap, aonde o conteúdo será quebrado para o próximo espaço livre.

O comportamento de para onde este conteúdo será deslocado vai depender dos eixos. A W3C recomenda o uso de uma junção de flex-wrap + flex-direction: o flex-flow.


                <style>
                    div#container {
                        flex-flow: row nowrap;
                    }
                </style>
